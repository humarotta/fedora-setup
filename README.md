# Fedora Setup

A personal script to automate the setup of a new [Fedora Workstation] installation.

## Usage

Open a terminal and run the following command:

```bash
bash -c "$(curl -fsSL https://github.com/humarotta/fedora-setup/raw/main/setup.sh)"
```
